PMAT

发表于 2023-05-28 更新于 2023-07-23
PMAT is an efficient toolkit for mitochondrial genome assembly. It can correct three-generation sequencing data using [canu](https://github.com/marbl/canu) or [NextDenovo](https://github.com/Nextomics/NextDenovo) and then use [Newbler](https://evomics.org/learning/assembly-and-alignment/newbler/) to assemble it, and it will find the mitochondrial genome sequence and generate the mitochondrial genome structure.

PMAT

an efficient assembly toolkit for plant mitochondrial genome

PMAT is an efficient toolkit for mitochondrial genome assembly. It can correct three-generation sequencing data using canu or NextDenovo and then use Newbler to assemble it. Finally, PMAT can identify mitochondrial genome sequences and generate mitochondrial genome structures.

Installation

recommended installation method

wget https://github.com/bichangwei/PMAT/releases/download/v1.1.0/PMAT-1.1.0.tar.gz
tar -zxvf PMAT-1.1.0.tar.gz
cd PMAT/bin
chmod a+x PMAT
PMAT --help

Repuirement

When inputting HiFi reads, the following software are required:
  • blastn Needs to be installed in the PATH.
  • singularity
When inputting CLR or ONT reads, the following software are required:
  • blastn Needs to be installed in the PATH.
  • canu or NextDenovo Suggested to be installed in the PATH.
  • singularity

autoMito

One-step de novo assembly of mitochondrial genome. This command corrects the raw ONT/CLR data or uses the corrected data or HiFi for assembly directly. Based on the assembly result, automatically select seeds for extension and filter false positives to obtain an assembly map of the mitochondrial genome.

run PMAT autoMito --help view the usage guide for the all command.

Required arguments:
  -i INPUT, --input INPUT
                        input raw sequencing file
  -o OUTPUT, --output OUTPUT
                        output directory
  -st SEQTYPE, --seqtype SEQTYPE
                        sequencing platform(ONT/CLR/HiFi)
  -g GENOMESIZE, --genomesize GENOMESIZE
                        Please enter the genome size of the species, such as 1G, 1000M.

optional arguments:
  -h, --help            show this help message and exit
  -tk TASK, --task TASK
                        all/p1/ Default: all
                        all : De novo assembly including error correction for ONT/CLR data and no error correction for HiFi data
                        p1  : Import error-corrected ONT/CLR or HiFi data for direct assembly
  -cs CORRECTSOFT, --correctsoft CORRECTSOFT
                        Correcting software using nextDenovo or Canu. Default: nextDenovo
  -cp CANU, --canu CANU
                        Please provide the install path of canu.
  -np NEXTDENOVO, --nextDenovo NEXTDENOVO
                        Please provide the install path of nextDenovo.
  -cfg CORRECTCFG, --correctcfg CORRECTCFG
                        config file for nextdenovo correct
  -fc FACTOR, --factor FACTOR
                        Subset extraction of error-corrected ONT, CLR or HiFi data. Sampling ratio factor in 0-1. Default=1
  -sd SUBSEED, --subseed SUBSEED
                        Sampling set random number seeds, Default=6
  -bn BREAKNUM, --breaknum BREAKNUM
                        break long reads (>30k) with this. Default: 20000
  -ml MINOVERLAPLEN, --minoverlaplen MINOVERLAPLEN
                        set minimum overlap length. Default:40
  -mi MINIDENTITY, --minidentity MINIDENTITY
                        set minimum overlap identification. Default:90
  -cpu CPU              The number of threads. Default: 8
  -l MINLINK, --minLink MINLINK
                        Filter according to the minimum link depth provided by the user
  -v, --version         show program's version number and exit

notes:

  1. Make sure blastn is installed in the PATH.
  2. -tk : There are two options for this parameter, all and p1. For ONT or CLR raw data, all is required, and it can be used to correct errors and trim the raw data. For error corrected data, you can set p1 to go directly to the assembly step. For HiFi data, this parameter can be ignored.
  3. -cs : If use ONT or CLR raw data, users need to provide the -cs parameter to select the error correction software, default use canu.
  4. -cp : When using canu for error correction, users need to use -cp parameter to specify the installation path of canu. This parameter can be ignored when canu is added to the PATH.
  5. -np : When using NextDenovo to correct errors, users need to use the -np parameter to specify the installation path of NextDenovo. In addition, you need to use canu to prune the data after NextDenovo error correction, so the -cp parameter is needed to specify the installation path of canu. When NextDenovo and canu are added to the PATH, this parameter can be ignored.
  6. -cfg : When using NextDenovo error correction, the user needs to specify a config file with -cfg. The cONTents of the configuration file are recommended to check NextDenovo. Also, it is recommended to add the -b parameter for correction_options in the config file.
  7. -fc : This parameter can randomly select a subset of sequencing data for error correction and assembly. The default is using all data.
  8. -sd : The seed used when extracting a subset of sequencing data that can be used with -fc.
  9. -bn : Assembly requires reads of length less than 30000, recommended to use the default parameters.
  10. -ml : Parameters used for assembly, default setting is 40. Recommended setting range is 40-200.
  11. -mi : Parameters used for assembly, default setting is 90. Recommended setting range is 90-95.
  12. -ml : Parameters used for filtering, recommended default settings.

graphBuild

If PMAT fails to generate the mitochondrial genome assembly map in one-step assembly, you can use this command to manually select seeds for assembly.

run PMAT graphBuild --help view the usage guide for the graphBuild command.

Required arguments:
  -c CONTIGGRAPH, --ContigGraph CONTIGGRAPH
                        PMATContigGraph.txt: a file that can get all connections bewteen contigs.
  -a ALLCONTIGS, --AllContigs ALLCONTIGS
                        PMATAllContigs.fna: a file that can get all information of contigs.
  -o OUTPUT, --output OUTPUT
                        output directory
  -gs GENOMESIZE, --genomesize GENOMESIZE
                        Please enter the genome size of the species, such as 1G, 1000M.
  -rs READSIZE, --readsize READSIZE
                        The read size or file for assembly, such as 5G or assembly_seq_subset.0.1.fasta.

optional arguments:
  -h, --help            show this help message and exit
  -cpu CPU              The number of threads. Default: 8
  -s SEEDS [SEEDS ...], --seeds SEEDS [SEEDS ...]
                        ContigID for extending. Multiple contigIDs should be separated by space. For example: 1 312 356
  -l MINLINK, --minLink MINLINK
                        Filter according to the minimum link depth provided by the user
  -v, --version         show program's version number and exit

notes:

  1. Make sure blastn is installed in the PATH.
  2. -c : PMATContigGraph.txt generated by autoMito command.
  3. -a : PMATAllContigs.fna generated by autoMito command.
  4. -gs : The genome size of the species.
  5. -rs : The amount of data used by the assembly, or provide assembly_seq_subset.0.1.fasta generated by the graphBuild command.
  6. -s : Manually select the seeds for the extension, it is recommended to use more than 3 seeds. Use spaces to split between different seed ids, e.g. 1,312,356.
  7. -l : Parameters used for filtering, recommended default settings.

# Examples of de novo assembly of mitochondrial genomes using Arabidopsis thaliana's publicly available HiFi data.
PMAT autoMito -i CRR302668.fastq -o ./test -st hifi -g 125m -fc 0.1

# Manual selection of seeds to generate assembly graphs.
PMAT graphBuild -c assembly_result/PMATContigGraph.txt-a ./test/assembly_result/PMATAllContigs.fna -gs 125m -rs ./test/subsample/assembly_seq_subset.0.1.fasta -o ./test_gfa

Output files

PMAT will produce a series of folders with the results:

  • ./test/subsample/assembly_seq_subset.0.1.fasta -> Data for assembly
  • ./test/subsample/assembly_seq.cut20K.fasta -> reads trimmed data
  • ./test/assembly_result/PMATAllContigs.fna -> The assembly result contains contig sequences
  • ./test/assembly_result/PMATContigGraph.txt -> The assembly result contains contig linking relationships
  • ./test/assembly_result/PMAT_raw.gfa -> Initial results of mitochondrial genome assembly for visualization
  • ./test/assembly_result/PMAT_master.gfa -> Mitochondrial genome assembly optimization results for visualization
